# My Portfolio

My personal portfolio built with React and TypeScript. Clean design, smooth animations, and a dark mode toggle.

## What's Inside

A simple portfolio showcasing my web development skills with:
- Responsive design that works (almost) everywhere
- Dark/light mode toggle
- Smooth animations with Framer Motion
- Scroll navigation between sections

## Tech Stack

- React 19 + TypeScript
- Tailwind CSS for styling
- Framer Motion for animations
- Lucide React for icons

## My Skills

**Frontend:** HTML, CSS, JavaScript, TypeScript, React, Angular, Vue, Tailwind CSS, Bootstrap

**Backend:** Node.js, PHP, Laravel, Symfony, Java, Python

**Tools:** Git, GitHub

## Quick Start

```bash
# Clone and run
git clone https://github.com/NoanWasTaken/portfolio.git
cd portfolio
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) and you're good to go!

## Build for Production

```bash
npm run build
```

Then deploy the `build` folder wherever you want.
